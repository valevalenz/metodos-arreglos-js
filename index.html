Métodos de Arreglos ¿Qué son?

Diferencia entre Función y Método

un bloque de código autónomo que puede ser llamado de manera independiente, mientras que un método es una función asociada a un objeto específico y generalmente opera sobre las propiedades de ese objeto.

Un objeto es

una colección de propiedades, y una propiedad es una asociación entre un nombre (o clave) y un valor. Los objetos se utilizan para almacenar múltiples

Un método de arreglo en JavaScript es

Un método que puedes usar para realizar diversas operaciones sobre los arreglos (arrays). Veremos una lista de los más comunes

Método splice()

Método versátil que se usa para eliminar o reemplazar elementos existentes y/o agregar elementos en una posición específica del arreglo

Saber índice

Método findIndex()

Devuelve el índice del primer elemento que satisface la condición (en forma de una función de retorno llamada callback). Devuelve -1 si no encuentra ningún elemento que la cumpla.
